# Django开发的天天新鲜项目

基于B2C的天天生鲜商城
> B2C(Business-to-Customer), 企业对个人的一种商业模式，简称"商对客". 


#### 技术栈
- 语言：Python3.5(Django=1.11)   
- 数据库: MySql、 redis  
- 任务队列(异步处理): celery(django-celery)     
- 分布式文件存储: FastDFS   
- 搜索引擎(商品检索)：  haystack(django-haystack)、whoosh、jieba   
- web服务器配置: Nginx+ uwsgi    
- 开发环境： PyCharm、Linux（ubuntu)
